# noncopyable
## 类
作用：noncopyable 被继承以后，派生类对象可以正常的构造和析构，但是派生类对象无法进行拷贝构造和赋值操作 
```cpp
#pragma once 

class noncopyable
{
public:
    noncopyable(const noncopyable&) = delete;
    noncopyable &operator = (const noncopyable&) = delete;
protected:
    noncopyable() = default;
    ~noncopyable() = default;
};
```
